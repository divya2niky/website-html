Start up project

For my start up project I am creating a personal portfolio. This web application enables students and engineers to showcase and share their portfolio of projects in a high-quality, 100% responsive design instantaneously in a single platform that helps them save time and reduce costs. It enables the users to show the audience their expertise, learnings, and experiences to help expand their network or increase their brand presence. Key features of my start up are: 1. Secure login over HTTPS 2. Ability to select the options Home, About, Projects, Contact, Settings and Dark/light mode 3. Display of choices 4. Ability to download the resume 5. Ability to send a message directly 6. Ability to access the github repo for the projects 7. Ability for the admin to create or delete questions.
